# Hits

## 简介

GitHub 仓库浏览计数，仅累计访问总次数（无去重）。

## 使用

仓库 README 中添加（请注意将 `b3log/solo` 换成自己的）：

```html
<a title="Hits" target="_blank" href="https://github.com/b3log/hits"><img src="https://hits.b3log.org/b3log/solo.svg"></a>
```
